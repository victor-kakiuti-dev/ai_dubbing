# Pipeline de Dublagem com Inteligência Artificial

Este projeto contém um **pipeline completo em Python** desenvolvido para automatizar o processo de **dublagem inteligente**.  
O sistema utiliza modelos modernos de **IA** para realizar:

1. **Transcrição automática** do áudio original.  
2. **Tradução** para o português utilizando modelos de linguagem avançados.  
3. **Sincronização** das falas traduzidas com o tempo original do vídeo.  
4. **Geração de áudio dublado**, preservando ritmo, intenção e temporalidade.

O objetivo é proporcionar um fluxo simples, escalável e automatizado para transformar qualquer áudio ou vídeo original em uma versão **dublada em português**, mantendo qualidade e precisão na adaptação.

O pipeline foi estruturado de forma modular, facilitando manutenção, evolução e integração com diferentes modelos de IA ou ferramentas de áudio.
